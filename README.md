# GCManager  
*A Risk of Rain 2 Mod manager featuring:*  

- ROR2MM uri support (you can install mods using the thunderstore's install buttons now!)  
  
- Mod Operations  
    - Installation  
    - Uninstallation  
    - Automatic updating  
    - Automatic dependency installation
